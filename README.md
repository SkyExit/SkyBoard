# SkyBoard

Installation:
- Download the SkyBoardSetup.exe from the [releases](https://github.com/SkyExit/SkyBoard/releases) Tab
- Install & Enjoy

Features:
- Over 200+ regular emojis
- Animal, Flower, Hand and Human emojis
- Click on emojis to copy them
- Copy confirmation
- Emoji click-history
